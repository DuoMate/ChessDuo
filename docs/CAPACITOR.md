# Capacitor Android — Runbook

## Overview

ChessDuo uses **Capacitor** as a thin WebView wrapper around the deployed web app at `https://chessduo-fe.onrender.com`. No Next.js build changes needed — the Android app loads the live web URL.

---

## Quick Start (Any Codespace / Fresh Machine)

```bash
npm run cap:setup    # One command — installs Java, Android SDK, Gradle, Capacitor, generates keystore
npm run cap:build    # Builds signed APK ready for sideload
```

That's it. The setup script detects what's already installed and skips completed steps. Safe to re-run.

## Manual Setup (If Script Fails)

### 1. Android SDK licenses

```bash
export ANDROID_HOME="$HOME/android-sdk"
export PATH="$ANDROID_HOME/cmdline-tools/latest/bin:$PATH"
echo -e "y\ny\ny\ny\ny\ny\ny\ny" | sdkmanager --sdk_root="$ANDROID_HOME" --licenses
```

### 2. Install SDK packages

```bash
sdkmanager --sdk_root="$ANDROID_HOME" "platform-tools" "platforms;android-34" "build-tools;34.0.0"
```

### 3. Install Capacitor

```bash
npm install --save @capacitor/core @capacitor/cli @capacitor/android
npx cap add android
```

### 4. Generate keystore

```bash
keytool -genkey -v -keystore chessduo.keystore -alias chessduo -keyalg RSA -keysize 2048 -validity 10000
```

### 5. Create signing properties

```bash
cat > android/keystore.properties << PROPS
storeFile=../../chessduo.keystore
storePassword=YOUR_PASSWORD
keyAlias=chessduo
keyPassword=YOUR_PASSWORD
PROPS
```

## Sideload to Phone

1. Download `app-release.apk` to your machine
2. Transfer to Android phone (USB, email, Google Drive, Bluetooth)
3. On phone: **Settings → Security → Install from unknown sources** (enable)
4. Open the `.apk` file → tap **Install**
5. Open ChessDuo → play!

## Troubleshooting

| Problem | Fix |
|---------|-----|
| `sdkmanager: command not found` | `export PATH="$HOME/android-sdk/cmdline-tools/latest/bin:$PATH"` |
| `ANDROID_HOME not set` | `export ANDROID_HOME="$HOME/android-sdk"` |
| `JAVA_HOME not set` | `export JAVA_HOME=$(dirname $(dirname $(readlink -f $(which java))))` |
| `keystore.properties: No such file` | Run `npm run cap:setup` to regenerate |
| `gradle: command not found` | `source "$HOME/.sdkman/bin/sdkman-init.sh"` |
| `License not accepted` | See Manual Setup → Step 1 |
| **Lost keystore password** | Cannot update app on Play Store. Generate new keystore, publish as new app |
| `app-release-unsigned.apk` instead of signed | Run `npm run cap:build` (injects signing config) |
| App crashes on launch | Check Render URL is live: `curl https://chessduo-fe.onrender.com` |
| Supabase auth loop in WebView | Ensure `androidScheme: 'https'` in capacitor.config.ts |

## Keystore — Critical Warning

`chessduo.keystore` is your app's **permanent identity**. If you lose it:
- ❌ Cannot update the app on Play Store
- ❌ Cannot publish bug fixes
- ❌ Must publish as a brand new app (lose all ratings, reviews, installs)

**Save it:** Email `chessduo.keystore` + password to yourself. Store in a password manager.
